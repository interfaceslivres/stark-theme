## Começando
Crie um repositório Git na sua máquina local com todas as alterações feitas durante o seu desenvolvimento e histórico de commits.

```bash
$ git clone https://github.com/interfaceslivres/stark-theme.git
```
```bash
$ cd ./stark-theme
```
